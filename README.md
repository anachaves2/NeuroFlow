# 🧠 NeuroFlow - Laboratório Cognitivo

**NeuroFlow** é um website interativo desenvolvido para a unidade curricular de Aplicações para a Internet I, focado em explorar conceitos da mente humana através de uma interface moderna e dinâmica. O projeto obteve a classificação máxima de **20/20**.

### 🚀 Funcionalidades Principais
* **Interface Imersiva**: Design baseado em *Dark Mode* e *Glassmorphism* utilizando HTML5 semântico e CSS3 avançado (Flexbox e Grid).
* **Jogos Cognitivos**: Implementação de lógica interativa em JavaScript para os jogos:
    * **Sudoku**: Desafio de lógica e posicionamento.
    * **Efeito Stroop**: Teste de interferência de cores e palavras.
    * **Teste de Reflexos**: Medição de tempo de resposta do utilizador.
* **Componentes React**: Formulário de contacto dinâmico desenvolvido em **React.js**.
* **Persistência de Dados**: Utilização de **LocalStorage** para gestão de estados e dados do utilizador.
* **Responsividade Total**: Experiência otimizada para Desktop e dispositivos móveis.

### 🛠️ Tecnologias Utilizadas
* **Frontend**: HTML5, CSS3, JavaScript (ES6+).
* **Framework**: React.js.
* **Ferramentas**: Git/GitHub para controlo de versões.

### 📊 Avaliação Técnica
* **Nota Final**: 20/20.
* **Destaque**: Excelente organização de código e implementação de lógica de jogo fluída.
